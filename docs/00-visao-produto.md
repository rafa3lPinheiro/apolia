# Visao do Produto

- Sistema web interno.
- Uso principal por operador interno.
- Pode existir mais de um usuario.
- Por enquanto, todos os usuarios possuem o mesmo nivel de acesso.

O sistema existe para organizar a operacao de planos de saude de varias empresas.

As tarefas centrais sao:

- manter empresas organizadas
- manter um plano por CNPJ
- manter beneficiarios atualizados
- gerar e acompanhar cobrancas
- registrar pagamentos manualmente
- enviar avisos
- controlar inadimplencia
- proteger e monitorar o caixa de emergencia da apolice

Objetivo pratico:

- reduzir retrabalho
- reduzir erro operacional
- dar visibilidade rapida para o operador
