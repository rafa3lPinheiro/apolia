# Cobranca: Ajustes

Nem toda cobranca seguira o valor padrao.

Casos esperados:

- juros por atraso
- multa
- desconto
- acrescimo eventual
- correcao manual

Por isso a cobranca deve guardar:

- valor_original
- valor_atual

E o sistema deve registrar o motivo da mudanca.

Ajuste nao pode apagar o historico do valor inicial.
