# Risco: Dados Sensiveis

Parte dos dados usados na analise de risco sao dados pessoais sensiveis.

Exemplos:

- dados de saude
- doencas pre-existentes
- informacoes sobre peso quando usadas para perfil de saude
- dados de menores de idade exigem cuidado reforcado

Implicacao:

- nao basta armazenar
- e preciso justificar finalidade
- e preciso limitar acesso
- e preciso proteger bem esses dados

Regra de produto:

- coletar apenas o necessario
- evitar texto livre desnecessario
- preferir classificacoes controladas
