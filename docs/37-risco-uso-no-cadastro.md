# Risco: Uso no Cadastro

Ao cadastrar um novo beneficiario, o operador deve ver a composicao atual da apolice.

Objetivo:

- entender o perfil atual
- estimar impacto de nova inclusao
- decidir com mais contexto

O sistema deve mostrar:

- total de titulares
- total de dependentes
- quantidade de idosos
- quantidade de criancas
- quantidade de casos com doenca pre-existente
- outros indicadores que forem aprovados na governanca de dados

Regra importante:

- o sistema informa
- o operador decide

Nao recomendado:

- impedir cadastro automaticamente
- reprovar beneficiario so por pertencer a uma categoria sensivel
