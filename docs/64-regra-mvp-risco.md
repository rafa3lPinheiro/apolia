# Regra: MVP de Risco

No MVP, o sistema deve comecar com o menor conjunto sensato de informacoes.

Entram no MVP:

- data_nascimento
- classificacao titular ou dependente
- dados necessarios para cobranca e operacao

Ficam fora do MVP por padrao:

- campo explicito de mulher em idade fertil
- campo explicito de acima do peso
- campo sensivel de saude sem necessidade validada

Doenca pre-existente:

- so deve entrar se houver necessidade operacional clara
- se entrar, deve seguir a regra de protecao reforcada
