# Fatura Tecnica: Reposicao

Quando o beneficiario inadimplente pagar, o valor usado do caixa de emergencia deve voltar.

Isso significa que o sistema precisa registrar dois momentos diferentes:

- saida do caixa de emergencia por inadimplencia
- retorno do valor para o caixa de emergencia apos pagamento

Sem essa reposicao, o saldo da empresa fica errado.

O sistema precisa deixar claro:

- quanto saiu
- quanto voltou
- qual saldo ainda esta comprometido
