# Fatura Tecnica: Uso

Se um beneficiario atrasar, a empresa pode usar o caixa de emergencia.

Objetivo:

- impedir que o atraso individual pause o plano coletivo

Quando isso acontecer, o sistema deve registrar:

- qual beneficiario causou o uso
- qual empresa foi afetada
- quanto foi utilizado
- quando foi utilizado
- por qual motivo

Essa utilizacao precisa ser rastreavel.

Ela nao pode existir apenas como saldo solto.
