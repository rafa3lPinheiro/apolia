# Principios Tecnicos

## Lean

- construir o minimo util
- priorizar o que reduz trabalho manual
- priorizar o que melhora a visibilidade
- evitar funcionalidade sem valor operacional direto

## Clean

- separar interface, aplicacao, dominio e infraestrutura
- manter regra de negocio fora da interface
- desacoplar cobranca, reserva e notificacao
- tratar auditoria como parte do sistema
- tratar dado sensivel com mais restricao que dado comum
- preferir derivacao de indicadores a persistencia de rotulos sensiveis
